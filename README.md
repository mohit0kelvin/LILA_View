# LILA BLACK - Player Journey Visualizer

A web-based visualization tool for exploring player behavior in LILA BLACK, an extraction shooter game. Built for the Level Design team to understand how players navigate maps, where fights happen, and which areas get ignored.

## Live App
🔗 **[https://lilaview.streamlit.app](https://lilaview.streamlit.app)**

## Tech Stack
- Python + Streamlit (web app)
- Plotly (interactive maps)
- Pandas + PyArrow (data processing)
